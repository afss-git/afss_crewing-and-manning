# Default Avatar Images

This directory contains default avatar images for different user types:

- `default-admin-avatar.jpg` - Default avatar for admin users
- `default-seafarer-avatar.jpg` - Default avatar for seafarer users  
- `default-shipowner-avatar.jpg` - Default avatar for shipowner users
- `default-user-avatar.jpg` - Default avatar for generic users
- `vessel-placeholder.jpg` - Default image for vessels

## Usage

These images are used as fallbacks when user profile images are not available.
Replace these placeholder files with actual default avatar images for your application.
